# Meshing-Handbook

An interactive, single-page reference document for **Meshing in ANSYS Workbench** — built entirely with vanilla **HTML, CSS, and JavaScript**.

**FEA • CFD • Mesh Quality • Convergence • Optimization**

**Live Website →** https://srikeatht.github.io/Meshing-Handbook/

---

## Overview

This handbook consolidates **23 structured meshing sections** into a visually navigable engineering reference — covering everything from **basic geometry discretization and element types** to **mesh quality, inflation layers, convergence studies, and optimization strategies**.

Designed for **students, analysts, and simulation engineers**, the handbook provides a practical understanding of **meshing workflows in ANSYS Workbench** for both **FEA and CFD applications**.

## Features

* Interactive sticky sidebar navigation
* Scroll-progress indicator
* Section-by-section reveal animations
* FEA vs CFD meshing comparisons
* Structured vs Unstructured mesh explanations
* Mesh quality metric cards
* Mesh generation workflow diagrams
* Inflation layer and y+ guidance
* Contact, sizing, and topology repair references
* Zero dependencies — no frameworks, no build tools

---

## Tech Stack

* **HTML5 / CSS3 / Vanilla JavaScript**
* **Fonts:** Space Grotesk + Fira Code + Inter via Google Fonts
* **Intersection Observer API** for scroll animations
* **GitHub Pages** for deployment

## Usage

Clone the repository and open `index.html` directly in a browser — no server required.

```bash
git clone https://github.com/your-username/Meshing-Handbook.git
cd Meshing-Handbook
open index.html
```
